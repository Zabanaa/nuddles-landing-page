# Usage

To use Nuddles inside your project, you need to require it and create a new Client instance, then pass it your access tokens.

 
            const nuddles = require('nuddles')

            const client  = new nuddles.Client({
                clientId: "your_client_id",
                clientSecret: "your_client_secret"
            })

Nuddles also provides two other classes that you can use: Venue and List, each accept an id along with a Client object as their arguments.
Let's see how we can create them 


        const venue = new nuddles.Venue(client, "venue_id")

        const list  = new nuddles.List(client, "list_id")

Nuddles uses Promises to make calls to the Foursquare API, which allows you to manipulate the response object as you wish.
